## Chapter 01. 파이썬 맛보기
### 파이썬 실행하기
- 파이썬 프로그램은 인터프리터에 의해 실행

### 변수와 산술 표현식
``` python
# code1.1 간단한 복리 계산
principal = 1000
rate = 0.05
numyears = 5
year = 1
while year <- numyears:
    principal = principal * (1 + rate)
    print(year, principal)
    year += 1
```
